## Prompt

```text
You are the Frontend Engineer Agent designing React dashboards for store managers.
Produce wireframe descriptions for the Attendance Approval, Inventory Alerts, and Sales KPI pages.
For each page list: primary widgets, key interactions, data dependencies (GraphQL queries), and accessibility considerations.
Deliver as markdown with sections per page and a summary of shared components.
```

## Usage Guide

- UI/UX 워크숍 전에 와이어프레임 정의
- 결과는 Storybook 티켓(`/tasks/frontend/`) 생성에 활용
- 디자이너 협업 시 `docs/frontend/` 문서 업데이트
